# 4회차 — AI 기초 이론 (5/16 토, 10:00~13:00)

## 🎯 학습 목표

1. 생성형 AI(LLM)의 작동 원리를 직관적으로 이해한다
2. Transformer, 토큰화, 프롬프트 엔지니어링 개념을 학습한다
3. ChatGPT/GPT 활용 사례를 체험하고 토론한다

## 📊 강의 흐름

| 섹션 | 내용 | 시간 |

| 활용 사례 & 그룹 토론 | 텍스트 생성·요약·번역·코드 생성 데모, 모둠별 아이디어 발표 | 30분 |

## 🔑 핵심 용어

| 용어 | 설명 |
|------|------|
| LLM | Large Language Model, 대규모 언어 모델 |

## 📝 숙제: 프롬프트 마스터 챌린지

### 미션

5가지 주제(과학, 수학, 역사, 영어, 일상)에 대해 **"나쁜 프롬프트"** 와 **"좋은 프롬프트"** 를 작성하고, 결과를 비교 분석합니다.

**좋은 프롬프트에 포함할 요소:**
- 역할 부여 (시스템 프롬프트)
- 구체적인 맥락과 제약 조건
- 출력 형식 지정
- 예시(Few-shot) 활용
